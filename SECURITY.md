# Security Policy

## Supported scope

Security reports are accepted for:
1. `agent_idp_service`
2. `login_consent_app`
3. Hydra integration scripts and deployment manifests in this repository

## Reporting a vulnerability

Please do not open public issues for vulnerabilities.

Report privately by emailing the maintainer address associated with this repository account, and include:
1. Affected component and version/commit
2. Reproduction steps
3. Impact assessment
4. Suggested remediation if available

## Response expectations

1. Initial acknowledgement target: 72 hours
2. Triage and severity assessment target: 7 days
3. Fix timeline depends on severity and complexity

## Disclosure

Coordinated disclosure is preferred. Public disclosure should wait until a fix or mitigation is available.
