# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:
1. Being respectful and constructive
2. Accepting feedback gracefully
3. Focusing on project outcomes over personal attacks

Examples of unacceptable behavior:
1. Harassment, discrimination, or intimidation
2. Trolling, insulting comments, or personal attacks
3. Publishing private information without permission

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior.

## Scope

This Code of Conduct applies in all project spaces, including issues, pull requests, and discussions.

## Enforcement

Report concerns to maintainers through GitHub issues marked as confidential/security-related where appropriate.

Maintainers may remove, edit, or reject comments, commits, code, and other contributions that violate this Code of Conduct.
